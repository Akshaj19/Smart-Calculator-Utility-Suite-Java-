# Smart Calculator & Utility Suite (Java)

A beginner-friendly Java console application that combines multiple calculator and utility operations in one menu-driven program.

## Features
- Find greatest of 3 numbers
- Addition
- Subtraction
- Multiplication
- Division
- Even or Odd check
- Swap two numbers
- Simple Interest calculation

## Requirements
- Java JDK 8 or above

## How to Run
1. Save the code as `SmartCalculatorUtilitySuite.java`
2. Open terminal or command prompt
3. Compile:
   javac SmartCalculatorUtilitySuite.java
4. Run:
   java SmartCalculatorUtilitySuite

## Learning Concepts
This project helps practice:
- Java basics
- Variables and data types
- Conditional statements
- Switch case
- Loops
- Scanner for user input
- Arithmetic operations

## Purpose
Perfect for beginners who want to strengthen core Java programming skills by building a practical mini-project.
